## 🕓 一、广告投放时间段优化

### 1️⃣ 不同品类的投放黄金时段

| 品类类型           | 推荐投放时段                  | 特征与原因           |
| -------------- | ----------------------- | --------------- |
| 电商产品（服饰、美妆、家居） | 10:00–13:00、19:00–23:00 | 上班休息 & 晚间娱乐购物高峰 |
| 游戏类            | 12:00–14:00、20:00–01:00 | 午休 & 晚间娱乐时间     |
| 教育/知识付费        | 09:00–12:00、20:00–22:00 | 工作/学习心态活跃时段     |
| 本地生活服务         | 11:00–14:00、17:00–21:00 | 外出/点餐/出行时间段     |
| 金融/保险类         | 09:00–18:00             | 工作时间内决策率高       |

### 2️⃣ 时间段优化策略

* **新计划前 3 天建议全天投放**，系统先积累数据。
* 找出高转化时段后，**设置“分时投放”**，保留 ROI 高的时段（如晚间 20:00–23:00）。
* 可以在“分时出价”中提高黄金时段 10–20% 出价，增加曝光。
* **避开无效时段**（如凌晨 2–6 点），尤其在转化成本较高的行业。

## 👥 二、广告人群定向优化

### 1️⃣ 定向策略思路

> “先广后精，再放量”

| 阶段   | 策略               | 说明                  |
| ---- | ---------------- | ------------------- |
| 初期测试 | 广泛定向（系统推荐、自动定向）  | 系统收集数据，了解潜在用户画像     |
| 优化阶段 | 精准定向（兴趣+行为+人群包）  | 利用巨量算数或后台人群洞察找到核心群体 |
| 放量阶段 | 相似人群扩量 Lookalike | 扩展相似用户，提升规模与ROI平衡   |

### 2️⃣ 常用定向参数（巨量引擎）

* **兴趣标签**：美妆、游戏、育儿、理财、健身等。
* **行为标签**：最近点击/浏览/下单/关注过某类内容。
* **地域定向**：高消费城市（如北上广深）or 本地转化优先。
* **年龄段**：

  * 美妆/服饰：18–30 岁
  * 教育/留学：25–40 岁
  * 金融/保险：30–55 岁
* **性别偏好**：

  * 女：美妆、服饰、母婴
  * 男：游戏、金融、数码

### 3️⃣ 数据回流与再营销

* **再营销人群包**

  * 已访问落地页但未下单的用户。
  * 已加购/咨询但未转化的用户。
  * 已成交用户（用于交叉销售）。
* **工具推荐**：巨量引擎「巨量云图」「人群洞察」工具。

## 🎨 三、用户画面（创意素材方向）

### 1️⃣ 素材核心要素

| 元素   | 关键技巧                                   |
| ---- | -------------------------------------- |
| 开头3秒 | 冲击性画面 / 提出痛点 / 展现结果（例：“原价199，现在只要39！”） |
| 主体画面 | 生活场景化，贴近用户身份（如办公室、家居、地铁）               |
| 人物呈现 | 真实感 > 高颜值；选模特时优先“像用户本人”                |
| 行动引导 | 明确 CTA（立即领取、了解更多、点我领取优惠券）              |
| 节奏控制 | 15s 视频中，4–5 次视觉刺激点（切换画面/字幕/音效）         |

### 2️⃣ 不同行业画面建议

| 行业   | 画面风格            | 文案/配音建议         |
| ---- | --------------- | --------------- |
| 电商   | 对比画面：旧产品 vs 新产品 | “用了这个才知道差距”     |
| 教育   | 老师讲解 + 学员反馈     | “上完这节课，孩子进步太快了” |
| 游戏   | 实录操作 + 爆点音效     | “上来就能打BOSS，太爽了” |
| 金融   | 办公场景 + 专业感      | “稳健理财，收益看得见”    |
| 本地生活 | 场景展示 + 口碑推荐     | “这家店真香！就在我家楼下”  |

### 3️⃣ 创意A/B测试建议

* 同时准备 3–5 条视频，分别测试：

  * **人物差异**（男女/年龄）
  * **镜头差异**（特写 vs 场景）
  * **文案差异**（优惠导向 vs 价值导向）
* 数据观察周期 ≥ 3 天，按 CTR 和 CVR 选优。
