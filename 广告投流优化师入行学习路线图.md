## 🧭 一、了解行业：投流优化师是做什么的？

**一句话定义：**

> 广告投流优化师（Ad Optimization Specialist）负责让广告“花得值”——通过数据分析、创意优化和出价策略，提升广告ROI（投资回报率）。

**核心工作内容：**

1. **广告投放**（抖音/巨量、快手、腾讯广告等平台）
2. **数据分析**（CTR、CPC、CVR、ROI）
3. **创意测试**（视频素材、封面、文案）
4. **人群定向与预算策略**
5. **转化追踪与再营销**
6. **与设计、运营、销售沟通，制定广告策略**

## ⚙️ 四、实操步骤（你要能“自己跑一条广告”）

以“抖音信息流广告”为例👇

### 步骤 1：账户搭建

* 登录 [巨量引擎后台](https://www.oceanengine.com/)
* 完善账户信息 → 充值预算 → 验证资质

### 步骤 2：创建广告计划

* 选择投放目标（表单收集、落地页转化、应用下载等）
* 设置预算与出价（可选 oCPC / CPM）
* 选择广告位（抖音、头条、西瓜视频）

### 步骤 3：定向与创意上传

* 选择地域、性别、兴趣、年龄
* 上传视频素材 + 文案
* 设置转化追踪（巨量引擎支持转化API与SDK）

### 步骤 4：测试与优化

* 一次性测试 5–10 条素材，3–5 种定向
* 保留高CTR & 高CVR的组
* 优化预算、时段、人群

### 步骤 5：复盘总结

* 做日报、周报
* 分析“高转化素材的特征”
* 制作素材优选池（Best Performer素材）

## 📈 五、优化技巧（进阶阶段）

### 🎯 投流技巧要点

| 环节   | 技巧                          |
| ---- | --------------------------- |
| 素材优化 | “前三秒抓眼球”，内容贴近真实生活场景         |
| 人群优化 | 广定向→系统推荐→再用精准标签             |
| 出价策略 | 新计划出价略高（抢量）→稳定后逐步降价控成本      |
| 分时投放 | 先全天→后筛选ROI高时段（晚间20–23点常见高峰） |
| 再营销  | 对点击未转化人群再次投放，提高转化率          |
| 报表复盘 | 每天关注CTR、CPC、CVR、CPA变化趋势     |
