## 🧩 一、基础设置篇：账户与结构优化

1. **账户结构清晰化**

   * 按“产品线—受众—创意类型”来搭建，例如：

     > 账户 → 计划组（兴趣/人群）→ 计划（投放目的/落地页）→ 单元（素材/文案）。
   * 优点：便于后期复盘数据、精准分析。

2. **预算分配**

   * 前期测试阶段：

     > 建议 5–10 条计划，每条预算为单日预算的 10–20%。
   * 后期放量阶段：

     > 可逐步增加预算，每次不超过 30%，防止系统重新学习。

3. **学习期策略**

   * 学习期建议保留 3 天以上的数据观察周期。
   * 在学习期不要频繁修改预算或素材，否则算法会重新学习，导致投放不稳定。

## 🎯 二、投放优化篇：素材、人群、出价策略

### 1️⃣ 创意素材优化

* **原则：高 CTR × 高 CVR = 高 ROI**
* 技巧：

  * 开头 3 秒决定成败：添加强刺激点（痛点、反转、惊讶画面）。
  * 素材建议周期：7 天迭代一次，留存高表现素材进入“优选池”。
  * 使用素材标签测试不同方向，如【情感】【夸张】【教程】【对比】。

### 2️⃣ 定向策略

* 建议“广撒 + 优化”：

  * 前期：用**自动定向（系统推荐人群）**。
  * 后期：逐步细分至**兴趣标签**、**自定义人群**。
* 人群扩量技巧：

  * 复制优质计划 → 放开性别/年龄 → 调整地域测试。

### 3️⃣ 出价与投放策略

* **转化目标广告（oCPM/oCPC）**

  * 起量慢但成本稳定。
  * 可搭配“放量期小幅提价法”：每次提价 10–15%，稳定放量。
* **ROI 导向投放**

  * 当有稳定转化数据时，可使用“ROI 优化出价”。

## 📈 三、数据分析与迭代优化

1. **关键指标监控**

   | 阶段   | 主要指标    | 优化方向       |
   | ---- | ------- | ---------- |
   | 曝光阶段 | CPM、CTR | 提升创意吸引力    |
   | 点击阶段 | CPC、点击率 | 优化标题、封面    |
   | 转化阶段 | CVR、CPA | 优化落地页与行动按钮 |

2. **A/B 测试思路**

   * 每次只改一个变量（素材、文案、定向）。
   * 保持测试周期 ≥3 天，以排除系统波动。

3. **落地页优化**

   * 加载时间 ≤3 秒（太慢影响转化）。
   * 首屏突出“价值点+行动按钮”。
   * 埋点追踪转化路径，结合巨量分析工具（巨量引擎-巨量云图）定位流失点。

## 🚀 四、进阶技巧

1. **巨量千川（电商投放）**

   * 利用“直播间 ROI 回流模型”：
     投放 → 引流直播 → 转化分析 → 精准再投放。
   * 重点指标：CTR > 1.5%，CVR > 3%。

2. **巨量算数 / 巨量创意中心**

   * 数据洞察：查看行业热度、竞品投放趋势。
   * 创意参考：查看同行视频素材（CTR 高的视频类型、文案风格）。

3. **自动化投放工具**

   * “自动投放+智能创意”可以快速测试多个素材方向。
   * 但要人工筛选数据，防止算法偏向低质量点击。
